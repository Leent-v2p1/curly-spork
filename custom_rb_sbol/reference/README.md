# Эталонные SQL-прототипы

В этой папке лежат эталонные SQL-прототипы витрин SBOL, собранные по бизнес-логике исходных Java-реализаций из `personalization-internal-sources/datamart/etl/erib`.

Назначение:
- зафиксировать ожидаемую логику витрин в компактном и читаемом виде;
- иметь опорный SQL для ревью миграции;
- запускать автоматическую сверку между эталоном и реализацией в `etl/workflows/*/dml/threads/*`.

## Состав
- `prototypes/*.reference.sql` — эталонные SQL-прототипы;
- `mapping.json` — карта сопоставления эталонов с реализованными workflow-thread SQL;
- `../tools/compare_reference.py` — скрипт сверки;
- `../tools/compare_reference.sh` — удобная оболочка для запуска сверки.
